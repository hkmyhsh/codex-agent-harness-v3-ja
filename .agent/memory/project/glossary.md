# 用語集

## Self-hosted Runner

GitHub ActionsのJobをユーザー管理環境で実行するRunner。

## queued

Workflow JobがRunner割り当てを待っている状態。

## ARC

Actions Runner Controller。Kubernetes上でGitHub Actions Runnerを管理する仕組み。

## MCP Gateway

AI Agentと外部Toolの間で、Tool実行・権限・監査ログを制御する境界。

## ABAC

Attribute-Based Access Control。役割だけではなく、環境、対象リソース、ブランチ、操作種別などの属性で制御する方式。
